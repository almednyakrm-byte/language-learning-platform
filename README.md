# منصة تعليمية لتعليم اللغات مع امتحانات تفاعلية
## نظرة عامة
منصة تعليمية لتعليم اللغات هي مشروع يهدف إلى توفير بيئة تعلم تفاعلية للطلاب لتعلم اللغات. يتميز المشروع بامتحانات تفاعلية وآليات تقييم متقدمة لتحسين مهارات الطلاب اللغوية.

## هيكل المشروع
هيكل المشروع كما يلي:
* `app`: يحتوي على كود التطبيق الرئيسي
* `config`: يحتوي على إعدادات التطبيق
* `database`: يحتوي على قاعدة البيانات
* `docker`: يحتوي على ملفات الدوكر
* `tests`: يحتوي على الاختبارات الوحدة

## تشغيل البيئة
لتشغيل البيئة، اتبع الخطوات التالية:
1. قم بتثبيت دوكر ودوكر كومبوز على جهازك.
2. قم بنسخ المشروع إلى جهازك.
3. انتقل إلى الدليل الرئيسي للمشروع.
4. قم بتشغيل الأمر `docker-compose up` لتشغيل البيئة.
5. انتظر حتى يتم تشغيل جميع الخدمات.

## مكونات المشروع
### الوحدات
* `lang`: وحدة لتعلم اللغات
* `exam`: وحدة لامتحانات تفاعلية
* `evaluation`: وحدة لتقييم الطلاب

### الجداول
* `students`: جدول يحتوي على معلومات الطلاب
* `languages`: جدول يحتوي على معلومات اللغات
* `exams`: جدول يحتوي على معلومات الامتحانات

### الأدوار
* `admin`: دور للمشرفين على المنصة
* `teacher`: دور للمعلمين
* `student`: دور للطلاب
